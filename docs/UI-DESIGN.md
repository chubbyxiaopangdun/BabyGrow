## 一、设计系统规范

### 1.1 色彩体系

```
主色（暖橙）            辅色（青绿）
--p:    #FF9B5E         --s:    #7ECFB3
--p-l:  #FFC9A0         --s-l:  #A8E6CF
--p-d:  #E87840         --s-d:  #5BB896
--p-g:  #FFF1E8         --s-g:  #EEF9F4

背景色系
--bg:    #FFFCF8  （主背景，温暖米白）
--bg-c:  #FFFFFF  （卡片背景）
--bg-s:  #FBF7F2  （次级背景）
--bg-h:  #F5EDE4  （悬浮/按下状态）

文字层级
--t:    #1C1411  （主文字，近黑暖色）
--t-2:  #6B5A4E  （次要文字）
--t-3:  #A8998E  （辅助/提示文字）
--t-i:  #FFFFFF  （反色文字）

语义色
--ok:   #5BB896  （成功/正常）
--warn: #F5A623  （警告/关注）
--err:  #E85C5C  （错误/危险）
--info: #4A9EF5  （信息/链接）
```

**色彩使用规则：**
- 主色 `--p` 用于：CTA按钮、选中态、品牌强调、关键数据
- 辅色 `--s` 用于：AI相关元素、睡眠模块、辅助操作
- 语义色用于：状态反馈（正常/警告/异常）、数据指标

### 1.2 字体层级

```
display     Noto Serif SC 700  28px  用于大标题、关键数字
H1          Noto Serif SC 700  22px  页面标题
H2          Noto Serif SC 700  18px  卡片标题、Modal标题
H3          Noto Sans SC 600   16px  区块标题
body        Noto Sans SC 400   14px  正文内容
caption     Noto Sans SC 400   12px  辅助说明、标签
tiny        Noto Sans SC 400   10px  底部导航文字
```

### 1.3 间距系统（4px基准）

```
--sp-1:  4px    极小间距（图标与文字）
--sp-2:  8px    小间距（紧凑元素）
--sp-3:  12px   中小间距（列表项内）
--sp-4:  16px   标准间距（卡片内边距）
--sp-5:  20px   中等间距（区块间）
--sp-6:  24px   大间距（大区块分隔）
--sp-8:  32px   超大间距（页面顶部留白）
```

### 1.4 圆角与阴影

```
圆角
--r-xs:  6px    标签、小按钮
--r-sm:  10px   输入框、次级卡片
--r-md:  16px   主卡片、弹窗
--r-lg:  22px   大卡片、底部弹窗
--r-xl:  30px   超大圆角元素
--r-full: 9999px  胶囊按钮、头像

阴影
--sh-sm:  0 1px 4px rgba(44,20,17,.06)     悬浮卡片
--sh-md:  0 4px 16px rgba(44,20,17,.08)    标准卡片
--sh-lg:  0 8px 32px rgba(44,20,17,.12)    弹窗浮层
--sh-glow: 0 4px 20px rgba(255,155,94,.30)  品牌辉光
```

### 1.5 动画规范

```
--ease-spring:  cubic-bezier(0.34, 1.56, 0.64, 1)  弹性效果
--ease-out:     cubic-bezier(0.16, 1, 0.3, 1)      出场动画
--ease-in:      cubic-bezier(0.4, 0, 1, 1)          入场动画

动画时长
即时反馈:  100ms   按钮点击、开关切换
标准过渡:  300ms   页面切换、列表展开
复杂动画:  400ms   弹窗出现、页面入场
延迟叠加:  50ms/项  列表项逐个入场(stagger)
```

## 二、页面架构总览

### 2.1 页面清单与层级

```
App
├── Welcome（首次引导）  ← 仅首次使用
├── Home（首页仪表盘）   ← 默认着陆页
│   ├── Schedule（日程详情）
│   ├── Sleep（睡眠记录）
│   ├── Health（健康记录）
│   ├── Growth（生长曲线）
│   └── Milestones（发育里程碑）
├── Records（成长记录）
├── AI Chat（AI对话）     ← Modal浮层
├── Profile（个人中心）
│   └── Settings（设置）
└── Modals
    ├── Child Selector（孩子切换）
    ├── Child Editor（添加/编辑孩子）
    ├── Feeding Form（喂养录入）
    ├── Sleep Form（睡眠录入）
    ├── Health Form（健康录入）
    └── Growth Form（生长数据录入）
```

### 2.2 底部导航（Tab Bar）

```
┌──────────────────────────────────────┐
│  🏠       📋      🤖      👶      ⚙️  │
│  首页     记录    AI助手   里程碑   我的  │
│  ●                         ○       │
└──────────────────────────────────────┘

Tab → Page 映射：
- 首页:   showTab('home')      → page-home
- 记录:   showTab('records')   → page-records
- AI助手: showTab('chat')      → 打开chatModal（全屏弹窗）
- 里程碑: showTab('milestones') → page-milestones
- 我的:   showTab('profile')   → page-profile
```

**设计要点：**
- 5个Tab，图标+文字，固定在底部，高度80px + safe-area
- 当前Tab高亮主色 `--p`，其余 `--t-3`
- AI Tab用辅色 `--s` 区分，图标为机器人
- Tab切换无动画跳转，直接切换页面内容

**交互逻辑：**
1. 输入出生日期后，自动计算月龄并显示（如"已满10个月"）
2. 性别默认选中"男"，点击切换
3. 关心问题多选，选中态为 `--p-g` 背景 + `--p` 边框
4. 点击"开始使用" → POST /children → 进入 Home 页
5. 页面从底部滑入动画，表单区域逐项延迟出现(stagger)

**数据校验：**
- 小名：必填，2-20字
- 出生日期：必填，不晚于今天
- 性别：必填，默认 male
- 喂养方式：必填

**模块拆解：**

| 模块 | 数据来源 | 交互 |
|------|---------|------|
| 孩子卡片 | state.currentChild | 点击→弹出Child Selector |
| 今日重点 | AI根据月龄+季节生成 | 每日自动更新 |
| 4顾问卡片 | 后端 /plan/today | 点击→进入Schedule页，筛选对应类型 |
| 记录快捷入口 | 静态配置 | 点击→跳转对应子页面 |
| 生成今日安排按钮 | GET /plan/today | 点击→生成+跳转Schedule |
| AI悬浮条 | 预设问题库 | 点击→打开AI Chat |

**状态设计：**
- 加载中：骨架屏(Skeleton)占位
- 无孩子：自动跳转Welcome
- 加载失败：显示重试按钮 + 缓存上次数据

## 八、V2.0 UI迭代方向

### 8.1 当前问题分析

| 问题 | 严重度 | 影响 |
|------|--------|------|
| 喂养/健康页部分仍在localStorage | P0 | 数据丢失风险 |
| 单文件3596行HTML，维护困难 | P1 | 开发效率低 |
| Chat是Modal不是Page，体验受限 | P1 | 无法查看历史、无独立导航 |
| 底部Tab只有5个，信息架构受限 | P2 | 新功能无处安放 |
| 图表用SVG内联，交互有限 | P2 | 无缩放、无tooltip |
| 无暗黑模式 | P2 | 夜间使用刺眼 |

### 8.2 V2.0 优化方案

#### A. 从Modal升级为Page

```
AI Chat: Modal → 独立页面(带消息历史、设置)
Settings: Modal → 独立页面(更丰富的设置项)
Child Editor: Modal → 独立页面(多步骤表单)
```

#### B. 5Tab → 6Tab导航(带FAB)

```
┌────────────────────────────────────┐
│  🏠    📋    ➕    🤖    📈    👤   │
│  首页  记录  添加  AI助手  统计  我的 │

│       │   │ FAB│   │               │  ← 中间凸起按钮
│       │   └───┘   │               │
└────────────────────────────────────┘

点击 ➕ 弹出半圆菜单:
  [+ 喂养] [+ 睡眠] [+ 健康] [+ 生长]
```

#### C. 首页信息密度升级

```
V1.0 首页: 顾问卡片(4宫格) + 记录列表
V2.0 首页: 时间轴视图(今天已做了什么/接下来做什么)
         + 快速录入浮动按钮
```

```
独立的统计分析页面:
```
