# BabyGrow · 产品执行文档（PM 验收标准）

> 更新日期：2026-05-01 | 状态：定稿
> 用途：产品经理验收各功能模块是否符合预期

---

## 一、验收总则

### 1.1 验收流程

```
开发完成 → 自测通过 → PM 验收 → Bug 修复 → 回归测试 → 上线
```

### 1.2 验收标准等级

| 等级 | 定义 | 处理 |
|------|------|------|
| **P0-阻塞** | 功能完全不可用 | 必须修复，阻塞上线 |
| **P1-严重** | 功能可用但体验极差 | 必须修复，不阻塞上线 |
| **P2-一般** | 体验有瑕疵 | 记录，下个迭代修复 |
| **P3-建议** | 优化建议 | 排入 backlog |

## 二、功能模块验收标准

### 模块 1：宝宝档案

| 验收项 | 标准 | 验收方法 |
|--------|------|---------|
| 创建宝宝档案 | 填写姓名/生日/性别/喂养方式后成功创建 | 实际操作创建 |
| 月龄自动计算 | 根据生日自动计算月龄，显示"已满 X 个月" | 修改生日验证 |
| 编辑宝宝信息 | 可修改所有字段并保存 | 实际操作编辑 |
| 切换孩子 | 多孩子家庭可切换当前孩子 | 创建 2 个孩子切换 |
| 数据持久化 | 关闭 App 重新打开数据不丢失 | 重启验证 |
| 输入校验 | 小名 2-20 字，生日不晚于今天 | 输入边界值测试 |

### 模块 2：AI 对话

| 验收项 | 标准 | 验收方法 |
|--------|------|---------|
| 问题分类 | 系统正确识别问题属于哪个领域 | 测试 10 个不同类型问题 |
| 个性化回复 | 回复包含宝宝名字、月龄等个性化信息 | 对话验证 |
| 多轮对话 | 支持上下文连续对话 | 连续问 5 个相关问题 |
| 响应速度 | 首字响应 < 3 秒 | 计时测试 |
| 安全审核 | 涉及医疗建议时附带"请咨询医生" | 测试发烧/用药问题 |
| 快捷问题 | 首页显示 4 个常用快捷问题 | 查看首页 |
| 打字动画 | AI 回复时显示打字指示器 | 观察动画 |
| 消息气泡 | 用户右对齐（橙色），AI 左对齐（白色） | 视觉检查 |

### 模块 3：日程管理

| 验收项 | 标准 | 验收方法 |
|--------|------|---------|
| 日程生成 | 根据月龄生成精确到 30 分钟的日程 | 生成并查看 |
| 时间轴显示 | 时间轴清晰，当前时间高亮 | 视觉检查 |
| 辅食时间 | 包含早餐/午餐/晚餐/加餐时间 | 核对日程 |
| 喂奶时间 | 包含每日 3-4 次喂奶时间 | 核对日程 |
| 睡眠时间 | 包含午睡+夜间睡眠时间 | 核对日程 |
| 活动时间 | 包含户外/早教/亲子时间 | 核对日程 |
| SOP 细节 | 每个时间段有具体操作指导 | 查看细节 |
| 天气联动 | 根据天气调整户外活动建议 | 修改天气测试 |

### 模块 4：辅食食谱

| 验收项 | 标准 | 验收方法 |
|--------|------|---------|
| 7 天食谱 | 生成连续 7 天的食谱 | 查看完整食谱 |
| 月龄适配 | 食物适合当前月龄（如 10 月龄无整颗坚果） | 核对月龄 |
| 过敏规避 | 自动排除过敏食物 | 设置过敏源验证 |
| 营养均衡 | 每日包含主食+蔬菜+肉类+水果 | 核对营养 |
| 食材清单 | 生成本周采购清单 | 查看清单 |
| 做法说明 | 每道菜有简要做法 | 查看做法 |
| 购买渠道 | 推荐购买平台（盒马/美团等） | 查看推荐 |

### 模块 5：睡眠管理

| 验收项 | 标准 | 验收方法 |
|--------|------|---------|
| 睡眠统计 | 显示平均时长/日均次数/夜醒次数 | 查看统计 |
| AI 分析 | 显示睡眠质量评分 + AI 建议 | 查看分析 |
| 记录列表 | 显示最近睡眠记录 | 查看列表 |
| 添加记录 | 可添加新的睡眠记录 | 实际添加 |
| 数据校验 | 结束时间 > 开始时间 | 输入错误时间测试 |

### 模块 6：发育里程碑

| 验收项 | 标准 | 验收方法 |
|--------|------|---------|
| 月龄显示 | 显示当前月龄大数字 | 查看页面 |
| 分类卡片 | 4 个分类（大运动/语言/认知/社交）+ 进度 | 查看卡片 |
| 里程碑列表 | 显示已达成（✅）和未达成（⬜） | 查看列表 |
| 打勾交互 | 点击可切换达成/未达成状态 | 实际点击 |
| 彩纸动画 | 打勾时触发彩纸爆发动画 | 观察动画 |
| 进度更新 | 打勾后分类进度实时更新 | 打勾后查看 |

### 模块 7：周末遛娃

| 验收项 | 标准 | 验收方法 |
|--------|------|---------|
| 城市识别 | 根据用户设置的城市推荐 | 切换城市测试 |
| 方案生成 | 提供室内+户外两种方案 | 查看方案 |
| 地点信息 | 包含地点名称/地址/费用/交通 | 核对信息 |
| 天气联动 | 根据天气推荐室内或户外 | 修改天气测试 |

### 模块 8：个人中心

| 验收项 | 标准 | 验收方法 |
|--------|------|---------|
| 信息展示 | 显示宝宝头像/名字/月龄/性别 | 查看页面 |
| 编辑信息 | 可修改宝宝信息 | 实际编辑 |
| 切换孩子 | 可切换/管理多个孩子 | 切换测试 |
| 城市设置 | 可设置所在城市 | 修改城市 |
| 数据导出 | 可导出宝宝数据 | 导出测试 |
| 清除数据 | 可清除所有数据（需二次确认） | 清除测试 |

## 三、UI/UX 验收标准

| 验收项 | 标准 | 验收方法 |
|--------|------|---------|
| 配色一致 | 使用柔和薄荷绿/桃/天蓝配色 | 视觉检查 |
| 字体规范 | 标题 Noto Serif SC，正文 Noto Sans SC | 检查字体 |
| 触控区域 | 所有可点击元素 ≥ 44px | 测量检查 |
| 动画流畅 | 页面切换无卡顿，动画 60fps | 观察动画 |
| 适配性 | 适配 iPhone SE ~ iPhone 15 Pro Max | 多设备测试 |
| 深色模式 | 支持系统深色模式（V1.5） | 切换测试 |

## 四、性能验收标准

| 指标 | 标准 | 测试方法 |
|------|------|---------|
| 首屏加载 | < 2 秒 | 计时测试 |
| AI 响应 | 首字 < 3 秒，完整回复 < 10 秒 | 计时测试 |
| 页面切换 | < 300ms | 计时测试 |
| 内存占用 | < 200MB | 性能监控 |
| 崩溃率 | < 0.1% | 崩溃统计 |

## 五、验收 Checklist

```
□ 宝宝档案：创建/编辑/切换/持久化
□ AI 对话：分类/个性化/多轮/速度/安全
□ 日程管理：生成/时间轴/SOP/天气联动
□ 辅食食谱：7天/月龄/过敏/清单/渠道
□ 睡眠管理：统计/分析/记录/校验
□ 发育里程碑：月龄/分类/打勾/动画/进度
□ 周末遛娃：城市/方案/信息/天气
□ 个人中心：展示/编辑/切换/导出/清除
□ UI/UX：配色/字体/触控/动画/适配
□ 性能：加载/响应/切换/内存/崩溃
```
