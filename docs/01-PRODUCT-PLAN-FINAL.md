# BabyGrow · 产品规划文档（定稿 v1.0）

> 更新日期：2026-05-01 | 状态：定稿
> 产品定位：AI 多 Agent 个性化育儿助手，替代小红书育儿攻略

---

## 一、产品愿景

**一句话定位：** 不用再刷小红书了——告诉 BabyGrow 你宝宝的情况，它直接给你定制化的方案。

**核心价值主张：**
1. **替代小红书** — 覆盖育儿全场景攻略，AI 直接定制化输出
2. **AI 育儿团队** — 多 Agent 专业协作，不是一个 AI 回答所有问题
3. **一个人带娃也不慌** — SOP 化日程管理，帮你安排好每一分钟
4. **越用越懂你** — 持续学习宝宝特点，建议越来越精准
5. **城市专属** — 杭州盒马买菜、上海山姆采购、周末遛娃地图

---

## 二、目标用户

| 用户画像 | 特征 | 核心需求 |
|---------|------|---------|
| **新手妈妈** | 0-12月龄宝宝，第一次当妈 | 不知道怎么带，需要 SOP 指导 |
| **一个人带娃** | 全职妈妈/爸爸独自带娃 | 时间管理、效率优化 |
| **职场妈妈** | 产假结束回归工作 | 高效利用碎片时间 |
| **隔代带娃** | 爷爷奶奶/外公外婆带 | 需要简单易懂的指导 |
| **二胎家庭** | 有两个不同月龄宝宝 | 个性化方案，不能一刀切 |

## 三、MVP 版本（V1.0）

### 3.1 MVP 范围

| 模块 | 功能 | 优先级 | 状态 |
|------|------|--------|------|
| **核心对话** | AI 育儿问答（多 Agent） | P0 | 待开发 |
| **宝宝档案** | 创建/编辑宝宝信息 | P0 | 已有基础 |
| **日程管理** | 每日 SOP 生成 | P0 | 待开发 |
| **辅食食谱** | 个性化 7 天食谱 | P0 | 待开发 |
| **食材采购** | 采购清单 + 购买渠道 | P1 | 待开发 |
| **睡眠建议** | 睡眠问题诊断 + 方案 | P1 | 已有基础 |
| **发育追踪** | 里程碑打勾 + 评估 | P1 | 已有基础 |
| **健康咨询** | 症状分级 + 就医建议 | P1 | 待开发 |
| **周末遛娃** | 城市化遛娃推荐 | P2 | 待开发 |

### 3.2 MVP 时间线

| 阶段 | 时间 | 交付物 | 负责人 |
|------|------|--------|--------|
| **Phase 1: 基础搭建** | 第 1-2 周 | 多 Agent 框架 + 宝宝档案 | 后端 |
| **Phase 2: 核心对话** | 第 3-4 周 | 调度 + 营养/睡眠/日程 Agent | 后端 |
| **Phase 3: 前端交互** | 第 5-6 周 | UI 开发 + 对话界面 | 前端 |
| **Phase 4: 知识库** | 第 7-8 周 | 育儿知识 RAG + 食谱库 | 后端 |
| **Phase 5: 联调测试** | 第 9-10 周 | 全链路测试 + 优化 | 全员 |
| **Phase 6: 上线** | 第 11-12 周 | MVP 发布 + 用户反馈 | 全员 |

### 3.3 MVP 成功指标

| 指标 | 目标值 | 衡量方式 |
|------|--------|---------|
| 日活跃用户（DAU） | 500+ | 后台统计 |
| 平均对话轮次 | 5+ 轮/次 | 对话日志 |
| 用户满意度 | 4.0+/5.0 | 应用评分 |
| 食谱采纳率 | 60%+ | 反馈数据 |
| 日程执行率 | 40%+ | 打卡数据 |

## 五、商业模式

| 收入来源 | 说明 | 预期占比 |
|---------|------|---------|
| **订阅制** | 基础免费 + Pro 月费（¥29.9/月） | 60% |
| **电商佣金** | 推荐商品 CPS 分成 | 25% |
| **品牌合作** | 母婴品牌广告/联名 | 10% |
| **增值服务** | 专家视频咨询（¥99/次） | 5% |

**Pro 会员权益：**
- 无限 AI 对话（免费版每日 10 次）
- 7 天个性化食谱生成
- 城市化遛娃推荐
- 辅食制作视频
- 历史数据完整保留

## 六、竞品对标

| 维度 | Huckleberry | 亲宝宝 | BabyGrow |
|------|------------|--------|----------|
| AI 能力 | 单一（睡眠） | 弱 | **多 Agent 协作** |
| 个性化 | 按月龄模板 | 无 | **基于宝宝实际数据** |
| 记忆系统 | 无 | 照片存储 | **长期记忆 + 持续学习** |
| 内容形式 | 文字 | 图片 | **文字 + 图片 + 视频** |
| 覆盖场景 | 睡眠 | 记录 | **全场景（替代小红书）** |
| 城市化 | 无 | 无 | **本地化服务** |
| 价格 | $12.99/月 | 免费+广告 | **¥29.9/月** |
